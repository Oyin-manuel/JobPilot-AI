# 🚀 JobPilot AI

An AI-powered career assistant that helps job seekers improve their resumes, evaluate job fit, and generate tailored cover letters using Google's Gemini AI.

---

## ✨ Features

### 📄 AI Resume Analyzer
- ATS Resume Score
- Overall Resume Score
- Recruiter's First Impression
- Strengths & Weaknesses
- Missing Keywords
- ATS Compatibility Analysis
- Priority Improvements
- Recommended Job Roles

---

### 🎯 AI Job Matcher
- Resume vs Job Match Score
- ATS Compatibility
- Matching Skills
- Missing Skills
- Missing Keywords
- Hiring Recommendation
- Resume Improvement Suggestions

---

### ✍ AI Cover Letter Generator
- Generates personalized cover letters
- Tailored to each job description
- Professional formatting
- TXT Download
- PDF Download

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/Oyin-manuel/JobPilot-AI.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application

```bash
streamlit run Home.py
```
